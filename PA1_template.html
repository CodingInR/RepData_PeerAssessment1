2) Fill in missing values in dataset. Approach: if all NAs for a day, replace with zero, else, replace NA with average of steps for the day.

remNA <- function(x,...) if (all(is.na(x))) 0 else mean(x,...)
datFilled <- transform(dat, steps = na.aggregate(steps, by = date, FUN = remNA))
